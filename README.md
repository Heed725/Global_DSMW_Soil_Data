# Global DSMW Soil Data

A repository containing nomenclature and reference information for the FAO Digital Soil Map of the World (DSMW).

## About

This repository provides standardized names and classifications for global soil data based on the FAO's Digital Soil Map of the World system.

## Contents

- Soil type names and classifications
- Reference data for DSMW soil units
- Standardized nomenclature for global soil mapping

## What is DSMW?

The Digital Soil Map of the World (DSMW) is a global soil database developed by the Food and Agriculture Organization (FAO) that provides comprehensive soil classification and mapping data at a global scale.

## Usage

This repository serves as a reference for:
- Soil classification studies
- Geographic information systems (GIS) applications
- Agricultural and environmental research
- Land use planning and analysis

## Source

Based on FAO's Digital Soil Map of the World framework.
